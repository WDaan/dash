## Installation

  
### Native
run
```
npm run build
```
place dist folder on your webserver

### With Docker
 [Docker Hub](https://hub.docker.com/repository/docker/wdaan/dash)

```
docker run -d --name dash -p 80:80 wdaan/dash
```
